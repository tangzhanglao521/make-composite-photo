# make-composite-photo

#### 介绍
本项目是用来合成图片的，主要使用了canvas，并且使用的是uniapp，可以做微信小程序以及app,h5等,如果对您有帮助请动动您可爱的小手。

#### 软件架构
vue,uniapp,canvas


#### 使用说明

下载hbuilder直接运行就可以
